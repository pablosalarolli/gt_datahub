# gt_datahub

`gt_datahub` é a biblioteca DataHub da GainTech para registro central de variáveis,
integração por conectores e resolução de bindings definidos por YAML.

## Baseline ativa

A referência normativa atual do projeto é a baseline `DataHub v3.7.2`.

## Estado atual

O projeto segue a ordem técnica definida no roadmap da baseline `v3.7.2`.
A fase 2 foi encerrada com catálogo, store e bootstrap de estado validados. O
próximo foco de implementação é a fase 3, dedicada ao YAML compiler e à
validação.
